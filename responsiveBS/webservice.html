What I do


Static website

A static website consists of a series of HTML files, each one representing a physical page of a website. On static sites, each page is a separate HTML file. When you visit the homepage, you are viewing only the actual homepage file.


Responsive Web Design

Responsive web design (RWD) or responsive design is an approach to web design that aims to make web pages render well on a variety of devices and window or screen sizes from minimum to maximum display size to ensure usability and satisfaction.


Customized layouts

Custom layout is the process of designing layout of electronic circuit manually with the help of layout editor. Electronic circuits can be Standard cells, Analog Circuits, Memory Circuits, IO Circuits and Mixed Signal Circuits.
